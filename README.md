# OS & System Programming: DOS Virtualization and Analysis

## 📌 Project Overview
This project focuses on the deployment, configuration, and analysis of a legacy operating system within a modern virtual environment. The primary objective is to understand low-level system interactions, memory management, and the historical foundations of modern operating systems.

## 🖥️ Project Scope: MS-DOS Virtualization
The core of this assignment involves the installation and technical evaluation of MS-DOS using VMware Workstation.

### Technical Highlights:
* Virtualization Layer: Configuration of hardware resources (RAM, Storage, CPU) to support legacy x86 architecture.
* Installation Workflow: Documented process of booting from virtual floppy disks and partitioning the virtual drive.
* System Analysis: * Examination of the Command Line Interface (CLI).
    * Understanding the FAT file system structure.
    * Analysis of core system files (IO.SYS, MSDOS.SYS, COMMAND.COM).

## 📂 Repository Contents
* Technical Report (PDF): Contains the full step-by-step documentation, screenshots of the installation and also virtualization concept.

## 🛠️ Environment & Tools
